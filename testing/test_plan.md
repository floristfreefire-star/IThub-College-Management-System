 План тестирования IThub Project Management

 Цели тестирования:
— Проверить удобство интерфейса системы
— Убедиться, что все основные функции работают
— Найти возможные ошибки и недочеты

 Типы тестирования:

 1. Функциональное тестирование
Что проверяется: основные функции системы
— Создание нового проекта
— Добавление участников в команду
— Назначение задач
— Отслеживание прогресса

Инструменты: Ручное тестирование
Среда: Windows 11, браузер Chrome

 2. Пользовательское тестирование  
Что проверяется: удобство для студентов
— Понятность интерфейса
— Легкость навигации
— Простота использования

Методы: Опрос одногруппников
